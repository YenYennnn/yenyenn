<h2>📊 Compliance & Data Analysis Projects:</h2>

- <b>📌 POS Weekly Compliance Dashboard</b>
  - <a href="#">View Project</a>
  <br/>
  <b>Problem:</b> Inconsistent reporting of agent transactions and charges made it difficult to detect discrepancies and enforce compliance.
  <br/>
  <b>Approach:</b>
  <ul>
    <li>Analyzed weekly transaction datasets (March Week 1 & 2)</li>
    <li>Built pivot tables to summarize transactions by agent</li>
    <li>Applied conditional formatting to flag mismatches</li>
    <li>Created visual dashboards for quick monitoring</li>
  </ul>
  <b>Key Insights:</b>
  <ul>
    <li>~15–25% of agents showed charge inconsistencies</li>
    <li>Higher discrepancies observed in high-value transactions (>₦50,000)</li>
    <li>Repeated issues traced to a small group of agents</li>
  </ul>
  <b>Impact:</b>
  <ul>
    <li>Improved compliance visibility across agents</li>
    <li>Enabled faster identification of discrepancies</li>
    <li>Reduced manual review time significantly</li>
  </ul>

- <b>📌 Raw Transaction Monitoring & Anomaly Detection</b>
  - <a href="#">View Project</a>
  <br/>
  <b>Problem:</b> Lack of structured monitoring made it difficult to detect unusual or potentially fraudulent transactions.
  <br/>
  <b>Approach:</b>
  <ul>
    <li>Analyzed raw transaction dataset</li>
    <li>Categorized transactions by amount bands and type</li>
    <li>Implemented rule-based anomaly detection (threshold & frequency checks)</li>
    <li>Used pivot tables for aggregation and trend analysis</li>
  </ul>
  <b>Key Insights:</b>
  <ul>
    <li>High-value transactions (>₦100,000) had higher irregularity rates</li>
    <li>Clusters of rapid transactions indicated potential risk behavior</li>
    <li>Transaction patterns varied significantly across agents</li>
  </ul>
  <b>Impact:</b>
  <ul>
    <li>Improved detection of suspicious transactions</li>
    <li>Provided structured approach to compliance monitoring</li>
    <li>Enhanced decision-making for audit and investigation</li>
  </ul>

- <b>📌 Agent Performance & Compliance Tracker</b>
  - <a href="#">View Project</a>
  <br/>
  <b>Problem:</b> No centralized view of agent performance and compliance behavior.
  <br/>
  <b>Approach:</b>
  <ul>
    <li>Integrated Agent Master List with transaction data</li>
    <li>Built performance metrics (volume, value, discrepancies)</li>
    <li>Designed dashboard with agent-level insights</li>

  <b>Impact:</b>
  <ul>
